* About Wanderer

Wanderer is an old text-based game originally written By Steven Shipway in c on on a sun/3-160

** Gameplay
In the following description, I will show the character used for each item when played in full-screen mode (by pressing '#').  This is also the character used in the screen files that run the game.
In wanderer, you are a spider (@) who must gather all the treasure (*) in a level and then go to the exit (X), all before the alloted number of moves expire.
You will encounter the following:
- Passable earth (:).  You get 1 point for each
- Solid rock (#) which cannot be blown up
- Solid rock (=) which _can_ be blown up
- A Teleport (T) which takes you to a new location (the new location is unknown in gameplay until you take the teleport, but in the screen files is denoted by 'A').  Using the teleport is optional, but you get a 50 point bonus if you do use it.
- Boulder (O).  Falls down.  Other boulders and arrows fall/slip around it.  A boulder falling on you kills you
- Left Arrow (<).  Shoots to the left.
- Right Arrow (>).  Shoots to the right.  Arrows can begin a level suspended and ready to shoot, but if you come near, they trigger and shoot.  Arrows shooting into you kills you.
- Instant Death Spot (!).  If you touch it, you die.
